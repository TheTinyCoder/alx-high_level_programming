The ``1-my_list`` module
========================

Using ``MyList``
-------------------

Importing:

	>>> MyList = __import__('1-my_list').MyList

Testing:

Empty list:
	>>> my_list = MyList()
	>>> print(my_list)
	[]
	>>> my_list.print_sorted()
	[]

Ordered list:
	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(4)
	>>> my_list.append(5)
	>>> print(my_list)
	[1, 2, 3, 4, 5]
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]
	>>> print(my_list)
	[1, 2, 3, 4, 5]

Unordered list:
	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.append(4)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(5)
	>>> print(my_list)
	[1, 4, 2, 3, 5]
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]
	>>> print(my_list)
	[1, 4, 2, 3, 5]

Unordered list with negative values:
	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.append(-4)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(5)
	>>> print(my_list)
	[1, -4, 2, 3, 5]
	>>> my_list.print_sorted()
	[-4, 1, 2, 3, 5]
	>>> print(my_list)
	[1, -4, 2, 3, 5]
